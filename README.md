## PineconePi Pilot是什么?
PineconePi Pilot(简称PPP）是由松果派开源硬件联合无人机爱好者一同开发的开源飞控程序。硬件基于松果派ONE（SWM320），
并推出多旋翼（主要是四旋翼）和固定翼程序。
